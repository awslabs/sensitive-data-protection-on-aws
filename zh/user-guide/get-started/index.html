    概述

    成功部署解决方案后，您可以访问控制台来检测敏感数据。

    • 第1步：连接数据源 添加 AWS 账户并创建数据目录。
    • 第2步：定义分类模板 通过管理数据标识符在模板中定义敏感数据。
    • 第3步：运行敏感数据发现作业 通过创建和管理数据发现作业来检测敏感数据。
    • 第4步：查看结果 通过数据目录可以看到数据源的元数据（如表结构）查看已更新的数据目录，并查看“摘要”页面检查仪表板。
