!!! Info "关于GDPR"
    《通用数据保护条例》（GDPR）是欧盟（EU）颁布的一项重要数据保护法规，它于2018年5月25日开始实施。GDPR旨在加强和统一所有欧盟成员国内个人数据（PII）的保护，同时也影响到欧盟外部的公司，只要它们处理欧盟公民的数据。
